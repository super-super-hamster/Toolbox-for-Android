# 工具箱

欢迎使用 **工具箱**！这是一款集成了多种实用生活工具应用程序，旨在为你的生活提供便捷。

---

## 核心功能详解

### 1. 📅 课程表
- **自定义排课**：支持管理 **20周** ,每天 1-5 节课的课程。
- **色彩**：每门课程会自动分配独一无二的颜色卡片，让你的课表一目了然。
- **导入课程**：在“设置 -> 导入课程表”中支持三种导入方式：
  1. **自然语言导入**：通过描述课程信息（包含课程名称、上课时间，可选地点和老师）进行导入（需要大模型 API KEY）。
  2. **JSON导入**：通过符合格式要求的 JSON 文本进行导入。
  3. **复制提示词**：让大语言模型生成符合格式的提示词，然后直接通过自然语言导入。

### 2. 🤖 助手
- **语音交流**：长按底部居中按钮呼出语音助手，松开即可发送语音进行对话。
- **快捷设备控制**：可以通过告诉助手你的需求，让它直接帮你打开“尺子”、“随机数”、“设置”等页面，或者询问它如何使用该应用。
- **热词识别**：在“设置 -> 热词”中添加热词，可以让助手更容易从语音中准确识别特定词汇。

### 3. 📏 尺子
- **测量**：支持屏幕上下滑动，最高可测量 **100 厘米** 长度的物品。
- **刻度**：提供标准的毫米级与半厘米级刻度线显示。
- **校准**：初始状态下，尺子刻度可能并不准确，你可以点击屏幕中心的“校准”按钮，修改放大倍数，与现实中的尺子进行校准。

### 4. 🎲 随机数生成器
- **滚轮选择器**：左右两侧的滚轮分别表示最小值和最大值（0 到 100），最终生成的随机数将介于其中。
- **滚动动画体验**：点击“生成”按钮后，最终停在你抽中的数字上。

### 5. 🎨 取色器
- **提取色彩**：点击加号选择图片，即可自动分析并提取出图片中的主要中等亮度颜色。

### 6. 📓 日记
- **便捷记录**：点击底部通用按钮即可创建日记，支持自由选择日期并设置标题。
- **隐私保护**：可在设置中开启“使用密码保护日记”，通过设备密码验证后才可进入。日记内容仅保存在本地，绝不会上传网络，保证隐私安全。

### 7. 🎚️ 分贝仪
- **噪音测量**：实时测量环境噪音分贝值。受硬件差异影响，测量结果仅供参考。
- **设备校准**：由于不同设备存在差异，点击底部通用按钮可进行校准。参考：安静环境（如隔音良好的图书馆）背景音约在 30 至 40 分贝。

### 8. ⏱️ 时间统计
- **统计**：每次进入时间页面将自动收集过去7天的所有应用使用时长，如果需要统计长时间的信息，请至多间隔7天打开一次时间统计页面。
- **展示**：在时间页面可以自由查看近12月、近30天和当天的应用使用情况统计，“近12月”和“近30天”中分别可以以热力图和折线图的方式直观了解使用情况。
- **显示与隐藏**：点击底部通用按钮可切换时间统计页面和设置可见性页面，在设置可见性页面可以隐藏不需要展示的应用。

### 9. 🌤️ 实时天气
- **展示**：在标题栏右上角直观展示气温和天气图标。
- **快捷查看城市**：点击天气图标，可将标题切换为你当前所在的城市及具体的文字天气状况（如“多云”）。
- **缓存**：天气数据会自动在本地缓存 5 分钟，并且 5 分钟后自动刷新。对于个人使用，和风天气提供的免费额度绰绰有余。

### 10. 🧭 悬浮底部菜单栏
- **全局多功能悬浮按键**：居中按钮长按可随时呼出语音助手，在特定页面（如课程表、热词设置）会动态切换为“添加”快捷操作。左右侧按键支持一键切出聊天弹窗及展开宫格菜单。
- **智能助手对话面板**：内置专属聊天界面，通过助手辅助使用该应用。
- **快捷导航**：提供快速前往各个核心应用模块。

## 📱 常见问题

**1. 智能助手配置**
使用助手功能前，请在“设置 -> 大模型 API”中填写 API Key。（当前仅适配 DeepSeek）
- **获取 API Key**：前往 [DeepSeek 开放平台](https://platform.deepseek.com/api_keys) 注册并创建。

**2. 天气功能配置**
如需使用天气查询功能，请前往“设置 -> 天气”中填写**和风天气**的 `API Key` 与 `API Host`。
- **获取 API Key**：
  1. 登录 [和风天气控制台](https://console.qweather.com/home)。
  2. 进入“我的项目管理” -> 点击“创建项目” -> 填写名称并保存。
  3. 点击“创建凭据” -> 填写凭据名称，身份认证方式选择 `API KEY`，保存后复制。
- **获取 API Host**：在 [和风天气控制栏设置](https://console.qweather.com/setting) 页面中复制。

**3. 课程表提醒配置**
如需开启课程表提醒功能，请在设置中指定“开学日期”。
- **提示**：将您所选日期的当周周一作为正式开学日。

**4. 版本**
安卓版本不低于8.0。
