# ToolBox (多功能工具箱)

欢迎使用 **ToolBox**！这是一款集成了多种实用生活工具应用程序，旨在为你的生活和学习提供便捷。

---

## 🌟 核心功能详解

### 1. 📅 课程表
- **自定义排课**：支持管理 **20周** ,每天 1-5 节课的课程。
- **直观编辑交互**：点击课表上的空白处即可快速“添加课程”；点击已有课程卡片，即可修改课程名称、教室、任课教师以及该课程的具体上课周次和节次，也可以便捷地将其删除。
- **色彩**：每门课程会自动分配独一无二的颜色卡片，让你的课表一目了然。
- 导入

### 2. 🤖 助手
- **语音交流**：长按底部居中按钮呼出语音助手，松开即可发送语音进行对话。
- **快捷设备控制**：可以通过告诉助手你的需求，让它直接帮你打开“尺子”、“随机数”、“设置”等页面，或者询问它如何使用该应用。
- **设置闹钟**：直接对助手说出提醒需求，AI 会自动为你设定系统闹钟功能。

### 3. 📏 尺子
- **测量**：支持屏幕上下滑动，最高可测量 **100 厘米** 长度的物品。
- **刻度**：提供标准的毫米级与半厘米级刻度线显示。
- **校准**：初始状态下，尺子刻度可能并不准确，你可以点击屏幕中心的“校准”按钮，输入缩放倍数，与现实中的尺子进行校准。

### 4. 🎲 随机数生成器
- **拟真滚轮选择器**：左右两侧的滚轮分别表示最小值和最大值（0 到 100），最终生成的随机数将介于其中。
- **滚动动画体验**：点击“生成”按钮后，最终停在你抽中的幸运数字上。

### 6. 🌤️ 实时天气
- **展示**：在标题栏右上角直观展示气温和天气图标。
- **快捷查看城市**：点击天气图标，可将标题切换为你当前所在的城市及具体的文字天气状况（如“多云”）。
- **缓存**：天气数据会自动在本地缓存 5 分钟，并且 5 分钟后自动刷新。对于个人使用，和风天气提供的免费额度绰绰有余。

### 7. 🧭 悬浮底部菜单栏
- **毛玻璃视效**：展开底部菜单、加载动画或语音输入时，背景会自动应用优雅的高斯模糊与毛玻璃效果（Haze Effect）。
- **全局多功能悬浮按键**：居中按钮长按可随时呼出语音助手，在特定页面（如课程表、热词设置）会动态切换为“添加”快捷操作。左右侧按键支持一键切出聊天弹窗及展开宫格菜单。
- **智能助手对话面板**：内置专属聊天界面，支持文本打字输入、消息历史漫游与精美拟物气泡，随时与助手深度互动。
- **宫格快捷导航**：提供快速前往课程表、尺子、随机数、游戏机、Tips 与设置等核心页面的入口。

## 📱 常见问题

**1. 智能助手配置**
使用助手功能前，请在“设置 -> 大模型 API”中填写 API Key。（当前仅适配 DeepSeek）
- **获取 API Key**：前往 [DeepSeek 开放平台](https://platform.deepseek.com/api_keys) 注册并创建。

**2. 天气功能配置**
如需使用天气查询功能，请前往“设置 -> 天气”中填写**和风天气**的 `API Key` 与 `API Host`。
- **获取 API Key**：
  1. 登录 [和风天气控制台](https://console.qweather.com/home)。
  2. 进入“我的项目管理” -> 点击“创建项目” -> 填写名称并保存。
  3. 点击“创建凭据” -> 填写凭据名称，身份认证方式选择 `API KEY`，保存后复制。
- **获取 API Host**：在 [和风天气控制栏设置](https://console.qweather.com/setting) 页面中复制。

**3. 课程表提醒配置**
如需开启课程表提醒功能，请在设置中指定“开学日期”。
- **提示**：将您所选日期的当周周一作为正式开学日。
